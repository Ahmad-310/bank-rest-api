# RESTful Bank API

This project is a simple RESTful API built using Java and Jersey, allowing you to:

- Add new customers
- Retrieve all customers
- Retrieve one customer by ID
- Update customer balances

## Technologies
- Java SE
- Apache NetBeans 17
- Jersey (JAX-RS)
- JSON
- Postman (for testing)
